## DangerIdentification Object

Represents a comprehensive hazard identification and risk assessment.

<ResponseField name="id" type="number" required>
  Unique identifier for the danger identification
</ResponseField>

<ResponseField name="danger" type="string" required>
  Name or brief description of the identified danger
</ResponseField>

<ResponseField name="current_defenses" type="string" required>
  Existing safety measures and controls in place
</ResponseField>

<ResponseField name="danger_area" type="string" required>
  Operational area affected by the danger
</ResponseField>

<ResponseField name="danger_type" type="string" required>
  Classification of the danger type
</ResponseField>

<ResponseField name="description" type="string" required>
  Detailed description of the danger
</ResponseField>

<ResponseField name="possible_consequences" type="string" required>
  Potential outcomes if the danger materializes
</ResponseField>

<ResponseField name="consequence_to_evaluate" type="string" required>
  Specific consequence selected for risk evaluation
</ResponseField>

<ResponseField name="root_cause_analysis" type="string" required>
  Analysis of underlying causes
</ResponseField>

<ResponseField name="information_source" type="InformationSource" required>
  Source of the hazard information
</ResponseField>

<ResponseField name="risk_management_start_date" type="Date" required>
  Date when risk management process began
</ResponseField>

<ResponseField name="analysis" type="Analysis" required>
  Risk analysis results
</ResponseField>

<ResponseField name="voluntary_report" type="VoluntaryReport">
  Associated voluntary report (if applicable)
</ResponseField>

<ResponseField name="obligatory_report" type="ObligatoryReport">
  Associated obligatory report (if applicable)
</ResponseField>

## InformationSource Object

Classification of how the hazard was identified.

<ResponseField name="id" type="string" required>
  Unique identifier for the information source
</ResponseField>

<ResponseField name="name" type="string" required>
  Name of the information source
</ResponseField>

<ResponseField name="type" type="string" required>
  Source type classification
  <Expandable title="values">
    - `PROACTIVO` - Proactive identification (audits, inspections)
    - `REACTIVO` - Reactive identification (incidents, accidents)
    - `PREDICTIVO` - Predictive identification (trend analysis, modeling)
  </Expandable>
</ResponseField>

## Analysis Object

Risk analysis evaluation results.

<ResponseField name="id" type="number" required>
  Unique identifier for the analysis
</ResponseField>

<ResponseField name="probability" type="string" required>
  Likelihood of occurrence (e.g., "FREQUENT", "OCCASIONAL", "RARE")
</ResponseField>

<ResponseField name="severity" type="string" required>
  Severity of consequences (e.g., "CATASTROPHIC", "HAZARDOUS", "MAJOR", "MINOR")
</ResponseField>

<ResponseField name="result" type="string" required>
  Overall risk level result (e.g., "HIGH", "MEDIUM", "LOW")
</ResponseField>
