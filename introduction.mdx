## Platform Modules

SIGEAC is organized into specialized modules for different aspects of aviation operations:

<Accordion title="Warehouse Management">
  Manage inventory of aircraft parts and components, track batches, handle incoming and outgoing articles, and monitor stock levels with automatic restock alerts.
</Accordion>

<Accordion title="Maintenance">
  Create work orders, assign technicians, track maintenance tasks, manage preliminary inspections, and generate maintenance reports for aircraft and components.
</Accordion>

<Accordion title="Purchasing">
  Process requisitions, create purchase orders, manage quotes from vendors, and track the complete procurement workflow from request to delivery.
</Accordion>

<Accordion title="Administration">
  Handle financial operations including credits, payments, cash management, bank accounts, and generate financial reports.
</Accordion>

<Accordion title="Safety Management System (SMS)">
  Report incidents, identify dangers, create mitigation plans, conduct safety activities, and maintain compliance with aviation safety regulations.
</Accordion>

<Accordion title="Planning">
  Schedule maintenance tasks, plan work orders, manage calendars, and coordinate resource allocation across operations.
</Accordion>

<Accordion title="Quality Control">
  Perform inspections, track quality metrics, ensure compliance with aviation standards, and maintain quality documentation.
</Accordion>

## Technology Stack

SIGEAC is built with modern web technologies:

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **UI Components**: Radix UI with Tailwind CSS
- **State Management**: Zustand and React Context
- **Data Fetching**: TanStack Query (React Query)
- **Forms**: React Hook Form with Zod validation
- **Real-time**: Laravel Echo with Pusher
- **Charts**: Recharts
- **PDF Generation**: React PDF and pdf-lib

## Getting Started

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install the required packages:
    
    ```bash
    git clone https://github.com/transmandu/SIGEAC_Frontend.git
    cd SIGEAC_Frontend
    npm install
    ```
  </Step>
  
  <Step title="Configure Environment">
    Set up your environment variables for API endpoints and authentication.
  </Step>
  
  <Step title="Run Development Server">
    Start the development server:
    
    ```bash
    npm run dev
    ```
    
    Open [http://localhost:3000](http://localhost:3000) in your browser.
  </Step>
</Steps>
